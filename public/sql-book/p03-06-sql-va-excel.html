1. Kết nối SQL với Excel phục vụ báo cáo tự động¶

1.1. From SQL server¶

  • Mở Excel >> Data >> From other sources >> From SQL server. Gõ tên địa chỉ Server names, user, password

  • Chọn Next liên tục đến khi hiển thị ra thông tin như ảnh dưới đây:

  • Chọn Finish để ra màn hình chọn Properties

  • Chọn Definition >> phần Command type chọn SQL >> gõ câu lệnh SQL truy vấn dữ liệu

Lưu ý: Có thể chọn Save password để khi gửi file cho client có thể tự refresh file khi dữ liệu trên SQL thay đổi.

1.2. Kết nối SQL với Microft Query¶

  • From other sources >> From Microsoft Query >> New Data source

  • Chọn tên đặt cho source kết nối, chọn driver là SQL server để đảm bảo client có thể refresh được dữ liệu.

  • Chọn Connect >> gõ tên Server muốn kết nối >> Bỏ chọn Use Trusted Connection để chọn User và Password. Chọn save Password cho những lần kết nối sau

  • Chọn All column trong phần Query Wizard

  • Chọn Next liên tục đến khi hiển thị ra đến phần chọn Return data to Microsoft >>  Finish.

  • Phần Properties >> Command text gõ câu lệnh SQL truy vấn, phần parameter cần để =?

  • Điền parameter value, lựa chọn vị trí ô để gõ dữ liệu filter >> chọn ô User this value và Refresh automatically.

1.3. Chỉnh sửa cậu lệnh truy vấn SQL sau khi đã tạo connection¶

  • Chọn: Data >> Connections >> Properties >> Definition để sửa câu lệnh truy vấn ở phần Command text.
Lưu ý: Sử dụng kết nối SQL và Excel sẽ cho phép tạo các báo cáo động trong Excel khi làm việc với SQL
