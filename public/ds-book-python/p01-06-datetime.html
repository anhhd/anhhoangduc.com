3. Dữ liệu thời gian trong Python

Trong Python, cấu trúc của datetime tương tự như R. Package numpy cũng cho phép làm việc với datetime hiệu quả.

[1]:

import numpy as np
my_date = np.array('2019-01-02', dtype = np.datetime64)
my_date

[1]:

array('2019-01-02', dtype='datetime64[D]')

[2]:

my_date + np.arange(5)

[2]:

array(['2019-01-02', '2019-01-03', '2019-01-04', '2019-01-05',
       '2019-01-06'], dtype='datetime64[D]')

Tuy nhiên, trong pandas, cấu trúc dữ liệu của datetime sẽ tương ứng với lubridate của R nhất. Xem các ví dụ sau

[3]:

import pandas as pd
my_date2 = pd.to_datetime('2019-01-01')
my_date2

[3]:

Timestamp('2019-01-01 00:00:00')

[4]:

my_date2.day_name()

[4]:

'Tuesday'

[5]:

# Thứ trong tuần
my_date2.dayofweek

[5]:

1

[6]:

# Extract tháng
my_date2.month_name()

[6]:

'January'

[7]:

# Extract tháng dạng số
my_date2.month

[7]:

1

Cộng trừ vector ngày

[8]:

my_date2 + pd.to_timedelta(np.arange(12), 'D')

[8]:

DatetimeIndex(['2019-01-01', '2019-01-02', '2019-01-03', '2019-01-04',
               '2019-01-05', '2019-01-06', '2019-01-07', '2019-01-08',
               '2019-01-09', '2019-01-10', '2019-01-11', '2019-01-12'],
              dtype='datetime64[ns]', freq=None)
