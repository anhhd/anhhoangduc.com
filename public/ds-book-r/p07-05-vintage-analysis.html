54  Vintage analysis

54.1 Giới thiệu

Vintage analysis là kỹ thuật phân tích giữa các nhóm khác nhau theo thời gian. Vintage analysis được sử dụng lần đầu khi so sánh chất lượng giữa các nhóm khách hàng bằng cách xem tỷ lệ nợ xấu của khách hàng sau cùng 1 khoảng thời gian. Ví dụ, tỷ lệ nợ xấu của khách hàng vay tín dụng sau 6 tháng, 12 tháng theo năm vay. Tuy nhiên, bên cạnh hoạt động quản trị rủi ro, Vintage Analysis còn có rất nhiều ứng dụng trong các hoạt động phân tích khám phá dữ liệu khác nhau như Marketing, Business Analaytics. Vintage Analysis có 2 loại cơ bản sau:

  • Phân tích tỷ lệ theo chiều thời gian: Đây là loại phân tích vintage gốc của quản trị rủi ro. Trong loại phân tích này, dữ liệu được chia thành các nhóm chỉ theo chiều thời gian. Chỉ số được đăng ký Ví dụ: Phân tích tỷ lệ convert của marketing theo tháng/quý đăng ký hồ sơ
  • Phân tích tỷ lệ dịch chuyển theo nhóm: Loại phân tích này được mở rộng từ loại hình phân tích gốc, cho phép phân tích không chỉ theo chiều thời gian mà còn nhiều chiều khác nhau. Xét về bản chất, phân tích vintage theo nhóm 1 chỉ là 1 trường hợp đặc biệt

Để thuận tiện, ta sẽ phân tích tập dữ liệu marketing như dưới đây.

library(dplyr)
library(ggplot2)
library(ggthemes)
library(lubridate)
library(scales)
load("data/marketing.rda")
marketing %>% head
marketing %>% summary

Trong tập dữ liệu trên, mỗi dòng ứng với 1 quan sát (khách hàng).

  • Biến start: Thời gian khách hàng đăng ký mua sản phẩm (hoặc nhận được thông tin quảng cáo, etc.)
  • Biến event: Thời gian khách hàng thực sự mua sản phẩm
analyze_vintage_group <- function(data, start, end, group1, group2, type = c("month", "day", "year")){
  group1 <- enquo(group1)
  group2 <- enquo(group2)
  start <- enquo(start)
  end <- enquo(end)
  result <- data %>%
    mutate(time = interval(!!start, !!end) %>% time_length(type)) %>% 
    group_by(!!group1, !!group2, time) %>% 
    summarise(no = n()) %>% 
    ungroup %>%
    group_by(!!group1, !!group2) %>%
    arrange(time) %>% 
    mutate(cumno = cumsum(no)) %>%
    mutate(total_no = sum(no)) %>%
    mutate(percent = cumno/total_no) %>%
    na.omit() %>% 
    ungroup
  return(result)
}

54.2 Phân tích vintage loại 1

Trong loại phân tích này, ta sẽ xem xét tỷ lệ chuyển đổi theo thời gian giữa các nhóm khách hàng, dựa vào thời gian đăng ký.

# Lấy tập dữ liệu test

df <- marketing %>% 
  filter(start <= event | is.na(event))

df <- df %>% 
  mutate(group = sample(c(1,2,3), size = nrow(.), replace = T)) %>% 
  mutate(month = floor_date(start, "months"))

result <- analyze_vintage_group(df, start, event, start, start, type = "day")
# Ví dụ khách hàng đăng ký vào ngày `20180823`
result %>% filter(start == ymd("2018-08-23"))

Với tập dữ liệu trên, ta có thể diễn giải như sau:

  • Có 47 khách hàng đăng ký mua sản phẩm vào ngày 2019-08-23
  • Ngay trong ngày đầu tiên đăng ký (time = 0), có 2.13% khách hàng mua sản phẩm
  • Sau 50 ngày (time = 50), có 10 khách hàng, tương ứng với 221.3% khách hàng mua sản phẩm
result %>% 
  ggplot(aes(time, percent)) +
  geom_line(aes(group = start), col = "darkgrey") +
  geom_smooth() +
  theme(legend.position = "none") +
  theme_light() +
  scale_color_tableau() +
  scale_y_continuous(labels = percent) +
  labs(x = "Days since registered", y = "Conversion rate",
       title = "Conversion rate of different registration time")

54.3 Phân tích vintage loại 2

Trong loại hình phân tích này, ta mở rộng loại hình phân tích, không chỉ xem xét yếu tố theo thời gian mà mở rộng toàn bộ phân tích theo nhiều nhóm khác nhau.

result_group <- df %>%
  analyze_vintage_group(start, event, group, month, type = "day")


result_group %>% 
  filter(group == 1 & month == ymd("20180901"))

result_group %>% 
  filter(group == 2 & month == ymd("20180901"))
result_group %>% 
  mutate(group = as.factor(group)) %>% 
  ggplot(aes(time, percent)) +
  geom_line(aes(col = group), size = 1) +
  theme(legend.position = "none") +
  theme_light() +
  scale_color_tableau() +
  facet_wrap(~month, scale = "free") +
  scale_y_continuous(labels = percent) +
  labs(x = "Days since registered", y = "Conversion rate",
       title = "Conversion rate of different registration time") +
  theme(legend.position = "top")
