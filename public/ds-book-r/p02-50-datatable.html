16  Biến đổi dữ liệu với data.table

16.1 Giới thiệu

data.table và dplyr là hai packages được sử dụng nhiều nhất trong R dùng để tính toán, biến đổi dữ liệu. Cả hai packages đều có ưu nhược điểm khác nhau.

Ngữ pháp:

  • Cấu trúc tổng quát của data.table: DT[i, j, by, ...] tức là: “Sử dụng data table DT, lọc các dòng với i, tính toán theo j, nhóm các biến theo by”. Các options khác có thể dùng …
  • dplyr sử dụng 3 nhóm câu lệnh cơ bản (filter(), arrange(), select(), mutate(), summarise()) và group_by(), tập trung vào việc dễ đọc, dễ hiểu

Hệ sinh thái:

  • dplyr phụ thuộc vào các package trong hệ sinh thái của tidyverse, chain operator phụ thuộc vào package magrittr

  • data.table là package đứng độc lập, có toán tử chain operator riêng

Tốc độ:

  • data.table có thể nhanh hơn dplyr nhiều lần khi làm việc với khối lượng dữ liệu lớn (lớn hơn 3 triệu dòng)

16.2 Các câu lệnh cơ bản

library(data.table)
library(dplyr)
df <- data.frame(
                v1 = 1:9,
                v2 = sample(c(1,2), 9, T),
                v3 = round(rnorm(3),2),
                v4 = letters[1:3])
data <- df %>% setDT(key = c("v1"))
data <- data %>% as.data.table() 
data %>% class
[1] "data.table" "data.frame"

16.2.5 .SD, SDcols

  • Hàm .SD cho phép tính toán nhanh 1 hàm nhất định vào nhiều cột
  • .SDcols cho phép liệt kê các cột cần được tính toán
data[, lapply(.SD, mean)]
   v1       v2        v3 v4   v2_new
1:  5 1.555556 0.1533333 NA 2.666667
data[, lapply(.SD, mean), .SDcols = c("v1", "v2")]
   v1       v2
1:  5 1.555556
  • Tổng hợp nhiều hàm với nhiều cột
data[, 
     c(lapply(.SD, mean),
       max = lapply(.SD, max)),
     by = v4,
     .SDcols = c("v1", "v2")]
   v4 v1       v2 v1 v2
1:  a  4 1.666667  7  2
2:  b  5 1.666667  8  2
3:  c  6 1.333333  9  2

16.3 dtplyr

dplyr mạnh hơn datatable về độ tường minh trong câu lệnh và có thể kết hợp với tidyverse. Trong khi đó, datatable lại nhanh hơn trong cấu phần xử lý số liệu. Để kết hợp ưu điểm của cả 2, ta có thể sử dụng packages dtplyr.

library(dtplyr)
library(dplyr)
library(data.table)

iris2 <- iris %>% lazy_dt()

iris2 %>% 
  group_by(Species) %>% 
  summarise(no = n(),
            mean_sepal = mean(Sepal.Length)) %>% 
  show_query()
`_DT1`[, .(no = .N, mean_sepal = mean(Sepal.Length)), keyby = .(Species)]
