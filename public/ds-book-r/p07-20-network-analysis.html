60  Network analysis

60.1 Giới thiệu

Network analysis(phân tích mạng lưới) là một nhánh kỹ thuật phân tích cho phép tìm sự liên kết giữa các đối tượng với nhau.

  • Các đối tượng trong một mạng liên kết được gọi là nodes hoặc vertices
  • Chiều quan hệ giữa các đối tượng được gọi là edge. Mối quan hệ giữa các node có thể là vô hướng (undirected) hoặc có hướng (directed)
library(tidyverse)
# library(ggraph)
library(tidygraph)
library(igraph)
library(visNetwork)

df <- data.frame(
  from = c(1, 2, 3),
  to = c(2, 1, 1)
)

node <- data.frame(id = c(1,2,3,4, 5)) %>% 
  mutate(label = "Node")

edge <- data.frame(
  from = c(1, 2, 3, 2),
  to = c(2, 1, 4, 3)
) %>% 
  mutate(label = "Edge")

visNetwork(node, edge) %>% 
  visEdges(arrows = "to") %>% 
  visLayout() %>% 
  visHierarchicalLayout(direction = "LR")

60.2 Các chỉ số cơ bản

Để xây dựng network, ta có thể sử dụng 2 dataframe chứa node và edge như sau.

node <- data.frame(id = c(1,2,3,4, 5, 6, 7, 8)) %>% 
  mutate(group = c("A", "B", "A", "A", "B", "A", "A", "A"),
         label = id) %>% 
  mutate(color = c("darkred", "darkred", NA, NA, NA, "darkred", NA, NA))

edge <- data.frame(
  from = c(2,3,2,4, 1, 3, 7, 8),
  to = c(4, 4, 3, 1, 1, 6, 6, 7))

network <- tbl_graph(
  nodes = node, 
  edges = edge,
  directed = T
)

visNetwork(node, edge) %>% 
  visEdges(arrows = "to") %>% 
  visOptions(highlightNearest = list(degree = 2,
                                     enabled = T,
                                     hover = T,
                                     algorithm = "hierarchical")) %>% 
  visHierarchicalLayout(direction = "LR")

60.2.1 Khoảng cách (distances)

Khoảng cách giữa các node được đo lường bằng số lần chuyển tiếp để có thể đi từ node này sang nốt khác. Khoảng cách này có thể được tính toán bằng hàm distances.

Lưu ý: Nên sử dụng algorithm = "johnson" để xác định chính xác khoảng cách ngắn nhất khi network là có hướng.

distances(network, algorithm = "johnson")
     [,1] [,2] [,3] [,4] [,5] [,6] [,7] [,8]
[1,]    0    2    2    1  Inf    3    4    5
[2,]    2    0    1    1  Inf    2    3    4
[3,]    2    1    0    1  Inf    1    2    3
[4,]    1    1    1    0  Inf    2    3    4
[5,]  Inf  Inf  Inf  Inf    0  Inf  Inf  Inf
[6,]    3    2    1    2  Inf    0    1    2
[7,]    4    3    2    3  Inf    1    0    1
[8,]    5    4    3    4  Inf    2    1    0

Kết quả trên đọc hiểu với node 2 như sau:

  • Node 2 (dòng thứ 2) mất 2 bước để có thể đi đến node 1 (2 > 4 > 1)
  • Node 2 không liên kết với node 5 nên khoảng cách giữa 2 node sẽ là Inf
  • Node 2 khoảng cách đến chính nó bằng 0

Bên cạnh đó, ta cũng có thể tìm được khoảng cách từ các node đến 1 node bất kỳ như sau

# Khoảng cách từ các node đến node 4
distances(network, to = 4, algorithm = "johnson")
